Research Interests

🛡️
Cyber Security
Network intrusion detection, threat intelligence, and advanced persistent threat (APT) mitigation strategies for enterprise and IoT environments.
🤖
Artificial Intelligence
Deep learning, hybrid neural architectures, and transformer-based models for security analytics and intelligent classification systems.
⚠️
Threat Mitigation
Botnet detection, DDoS attack prevention, and proactive threat response frameworks for modern networked infrastructures.
⚙️
Technology Risk Management
Enterprise security frameworks, authentication protocols, and federated learning for privacy-preserving threat detection in sensitive data infrastructures.
📡
IoT Security
Smart city network protection, satellite-integrated IoT security, and wireless sensor network (WSN) threat detection and prevention.
🔬
Machine Learning
Feature selection optimization, class balancing via SMOTE, and metaheuristic-tuned ELM and LSTM models for cybersecurity applications.
